# CareerCopilot
Find smarter. Apply better.

🚀 AI-powered career assistant.

CareerCopilot automatically discovers remote job opportunities across Europe, analyzes them using AI, scores them against your profile, and helps manage your job search.

## Features

- Multi-source job aggregation
- AI-powered job scoring
- Remote-first search
- Europe-wide opportunities
- Application tracking
- Dashboard
- Telegram notifications

## Tech Stack

- Python
- FastAPI
- PostgreSQL (Supabase)
- Streamlit
- Make
- OpenAI
